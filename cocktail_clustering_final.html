‚ùì The Problem We're Solving

Users exploring the cocktail cartography visualization face:

  • Choice Paralysis: 21 slider positions without clear differentiation
  • Unclear Purpose: What's the difference between position 7 and 8?
  • Abstract Clustering: Dots on a map without names or explanations

Our Goal: Identify which views actually matter, name the clusters clearly, and simplify the interface to guide users to meaningful discoveries.

üéØ Conclusions & Next Steps

The Big Picture

Cocktail similarity is inherently multi-dimensional. No single clustering perfectly captures all relationships. However, 5-6 well-chosen perspectives capture the vast majority of useful patterns, while 21 positions create confusion without adding value.

Immediate Actions:

  1. Replace the 21-position slider with 5-6 radio buttons
  2. Default to alpha=0 k=8 clustering
  3. Add descriptive names to all clusters
  4. Create tooltips explaining what each view reveals
  5. Highlight edge cocktails for users interested in innovation
